# OrbFall

OrbFall is a fast-paced arcade game inspired by "Breakout". Control your paddle, keep the orb in play, and shatter through a wall while chasing the highest possible score.

---

## Features

**Fast, explosive gameplay**  
  High-speed ball physics, satisfying impacts, and intense action from the first second.

**Highscore saving**  
  Your best runs are stored locally so you can always compete against yourself.

**Time slowdown for precision bouncing**  
  Temporarily slow down time to line up perfect rebounds and escape impossible situations.
